Webnovel to Epub

Context / Problem

Reading webnovels online is convenient, but having them available offline as EPUB files makes them easier to manage and read on e-readers. Manually copying chapters is tedious and slow. This project was built to automate that process entirely.

What I Built

A multithreaded Java application that scrapes web novel chapters from a source website and packages them into a properly formatted EPUB file, ready for any e-reader. It handles chapter ordering, metadata, and concurrent downloads to speed up the process.

What I'd Improve Next
