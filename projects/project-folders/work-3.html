Lamesa

Context / Problem

Our final project for Core Java 1 given by our professor. Lamesa was built as a group project to provide a desktop management system that centralizes these operations in one clean interface.

What We Built

A full desktop application built with Java and JavaFX, using SQLite as the local database and Maven for dependency management. The app includes screens for order management, menu editing, and table status tracking, all with a consistent UI built in JavaFX.
