Card Gallery

Context / Problem

Our professor wanted us to practice building layouts and card components using pure HTML and CSS, and let us choose a theme we want. A Will Eternal is one of my favorite webnovels, so I used its characters as the content for the gallery.

What I Built

A responsive card gallery featuring the main character of A Will Eternal and his wives. The layout uses CSS Grid and Flexbox for responsiveness. The project is live on Netlify and was one of my earliest pure HTML/CSS builds.
